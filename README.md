# String-Tokens-Problem
**void StringTokens(char[])** <br/>

Write a function which takes a string and prints words in it. For example:<br/>

String: I am a student of ITC in FAST-NU.<br/>

Tokens:<br/>
I<br/>
am<br/>
a<br/>
student<br/>
of<br/>
ITC<br/>
In<br/>
FAST-NU.<br/>

Hint: words are separated by spaces.
